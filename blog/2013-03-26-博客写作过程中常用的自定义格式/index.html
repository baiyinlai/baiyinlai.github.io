博客写作过程中常用的自定义格式

  • 学习与提升

本博客的所有文章均用 markdown 语言完成，在写作过程中，难免会遇到一些格式上的问题，因此将逐渐积累并整理一些常用的写作格式，比如代码片段、引用语等。

引用语的书写格式

两种格式：markdown 的格式和 html 的 <blockquote></blockquote> 标签格式。

当引用语没有引用出处时，直接利用 markdown 格式比较方便，在文本编辑器中直接输入一下内容即可：

> 这是引用语内容

输出结果为：

这是引用语内容

当引用语有出处时，要使用自定义的 html 标签 blockquote 样式，在文本编辑器中直接输入一下内容：

<blockquote>Life was like a box of chocolates, you never know what you're gonna get.<span class="quote_src">Forrest Gump</span></blockquote>

输出结果为：

Life was like a box of chocolates, you never know what you're gonna get. Forrest Gump

命令行工具名称、选项等特殊格式

使用 <kbd></kbd> 标签：

<kbd>emerge</kbd>

输出：

emerge
