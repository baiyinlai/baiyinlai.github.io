文件批量重命名 Bash 脚本

  • 2013-09-30
  • Baige5117
  • 学习与提升
  • 编程语言  脚本  批处理  

任务描述

将某一目录下的所有文件重命名为具有某种规则的名称，比如按数字升序的文件名。

脚本程序

#!/bin/sh

name=1
for f in `ls $1`
do
    mv "$1/$f" $2/$name
    ((name++))
done

程序解释

其实没啥解释的，主要用到 mv 命令来完成重命名。
